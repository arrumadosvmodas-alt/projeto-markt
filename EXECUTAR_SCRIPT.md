# 🚀 COMO EXECUTAR O SCRIPT E GERAR OS 12 SLIDES

## ⚡ PASSO 1: Abra o CMD/PowerShell (30 segundos)

### **Windows 10/11:**
1. Pressione: **Windows + R**
2. Digite: `cmd` ou `powershell`
3. Pressione **Enter**

---

## 📂 PASSO 2: Vá para a Pasta do Projeto (30 segundos)

Cole no CMD/PowerShell:

```bash
cd C:\Projeto Markt
```

Pressione **Enter**.

---

## 🐍 PASSO 3: Execute o Script Python (1 minuto)

Cole no CMD/PowerShell:

```bash
python gerar_slides_geribanca.py
```

Pressione **Enter**.

---

## ⏳ AGUARDE (5 MINUTOS)

O script vai rodar e você verá algo assim:

```
============================================================
🎬 GERADOR DE SLIDES GERIBANCA
============================================================

Criando Slide 1/12 - Apresentação do Problema...
✅ Slide 1 salvo: C:\Projeto Markt\GeribancaTikTok_Slides\Slide_01.png

Criando Slide 2/12 - Apresentação da Solução...
✅ Slide 2 salvo: C:\Projeto Markt\GeribancaTikTok_Slides\Slide_02.png

...

✅ TODOS OS 12 SLIDES CRIADOS COM SUCESSO!
============================================================

📁 Pasta de saída: C:\Projeto Markt\GeribancaTikTok_Slides\

🎬 Próximo passo: Importe os PNGs no CapCut Pro!
============================================================
```

---

## ✅ PRONTO!

Todos os 12 slides (PNG) estarão em:

**C:\Projeto Markt\GeribancaTikTok_Slides\**

- Slide_01.png
- Slide_02.png
- Slide_03.png
- ... (até Slide_12.png)

---

## 🎬 PRÓXIMO PASSO: Importar no CapCut Pro

1. Abra **CapCut Pro**
2. Crie novo projeto
3. Importe todos os 12 PNGs
4. Configure cada slide para **10 segundos**
5. Adicione:
   - Text-to-Speech em português (narração)
   - Transições
   - Música de fundo
6. Exporte e publique no TikTok!

---

## 🆘 PROBLEMAS?

### **"Python não foi encontrado"**
- Instale Python: https://www.python.org/downloads/
- Na instalação, **marque "Add Python to PATH"**
- Reinicie o CMD e tente novamente

### **"Pillow não foi encontrado"**
Cole no CMD:
```bash
pip install Pillow
```
Pressione Enter, depois rode o script novamente.

### **Altro erro?**
Mande a mensagem de erro que apareceu!
