## 📱 COMO MONTAR NO CAPCUT (GRÁTIS)

### Passo 1: Preparar os Slides
1. Crie 12 imagens (1080x1920px) representando cada bloco
2. Ou encontre designs similares online (Unsplash, Pexels)
3. Use ferramentas como Canva para customizar rapidamente

### Passo 2: Adicionar no CapCut
1. Abra CapCut Mobile (iOS/Android) ou Desktop
2. Create New → Import as Images
3. Selecione as 12 imagens em ordem
4. Defina duração de cada slide para 10 segundos
5. Menu → Aspect Ratio → 9:16 (vertical)

### Passo 3: Adicionar Textos Animados
1. Para cada slide, adicione o texto principal:
   - Bloco 1: "CONTROLE TOTAL"
   - Bloco 2: "GERIBANCA"
   - etc.
2. Use efeito "Type-on" ou "Fade-in"
3. Sincronize com o áudio

### Passo 4: Adicionar Áudio
1. **Narração:** Grave sua voz (ou use TTS do CapCut)
   - Settings → Text-to-Speech → Selecionar idioma Português
   - Cole o texto de cada bloco
   - Ajuste velocidade para ~1.0x
2. **Música:** Add Music → Busque no royalty-free
   - Coloque no fundo, reduza volume para 30-40%
3. **Efeitos:** Add Sound Effects conforme timecodes acima

### Passo 5: Transições e Efeitos Visuais
1. Entre cada slide, adicione transição:
   - Whip / Fade / Zoom (escolha uma e mantenha consistência)
   - Duração: 0.3s-0.5s
2. Adicione "Zoom" ou "Pulse" em elementos importantes (CTA, números)

### Passo 6: Revisar e Exportar
1. Preview completo (assista os 2 minutos inteiros)
2. Certifique-se que áudio está sincronizado
3. Ajuste cores com o efeito "Color Corrector"
4. Export → 1080x1920 (9:16)
5. Formato: MP4, 30fps
