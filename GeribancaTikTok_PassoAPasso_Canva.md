## 📌 PRÉ-REQUISITO: SETUP INICIAL (5 MINUTOS)

### Passo 1: Crie Novo Design
1. Abra **Canva.com** → Clique em "Criar"
2. Busque: **"Custom size"** (tamanho personalizado)
3. Digite: **Largura: 1080** | **Altura: 1920**
4. Clique "Criar design"

### Passo 2: Salve a Paleta de Cores
1. Na barra lateral esquerda, procure "**Paleta de cores**"
2. Clique no "+" para adicionar cores
3. Copie e cole cada cor (uma por uma):

```
#7C3AED   (Roxo primário - GeriBanca)
#06B6D4   (Ciano secundário - GeriBanca)
#10B981   (Verde sucesso)
#EF4444   (Vermelho alerta)
#0F172A   (Azul muito escuro - fundo)
#FFFFFF   (Branco texto)
#94A3B8   (Cinza neutro)
```

**Como adicionar cor:**
- Clique no "+" na seção de cores
- Paste o código HEX (ex: #7C3AED)
- Pressione Enter
- Repetir para cada cor

### Passo 3: Configure o Fundo
1. Clique no **fundo branco** (área em branco do design)
2. À direita, em "**Cor de fundo**", cole: **#0F172A**
3. Pronto! Fundo escuro aplicado

## 📱 SLIDE 1 - APRESENTAÇÃO DO PROBLEMA (0:00-0:10)

```
│      ❓ ❓ ❓             │

│   "Você Sabe Quanto      │

│   Controle Total da Banca│
│   Evite Perdas Desne...  │
└──────────────────────────┘
```

### SEQUÊNCIA DE CLIQUES NO CANVA:

**Passo 1: Adicione os Ícones de Interrogação**
1. Clique em "**Elementos**" (lado esquerdo)
2. Busque: "**question mark**" ou "interrogação"
3. Adicione 3 ícones na cor vermelha
4. Posição: Topo do slide, distribuídos (lado a lado)
5. Tamanho: Grande (~120px cada)
6. **Dica:** Para fazer 3 do mesmo tamanho, copie o primeiro (Ctrl+D duas vezes)

**Passo 2: Adicione Ícones de Dinheiro**
1. Clique em "Elementos"
2. Busque: "**money**" ou "💸"
3. Adicione 2-3 emojis de dinheiro
4. Posição: Abaixo dos interrogações, descendo
5. Tamanho: ~80px cada
6. Cor: Branco

**Passo 3: Adicione Divisor/Linha**
1. Clique em "Elementos"
2. Busque: "**line**" ou "divisor"
3. Selecione uma linha reta
4. Tamanho: ~200px comprimento, 2px altura
5. Cor: **#7C3AED** (roxo)
6. Posição: Meio do slide, horizontal

**Passo 4: Adicione Texto Principal**
1. Clique em "**Texto**" (lado esquerdo)
2. Clique em "**+ Adicionar texto**"
3. Type (copie e cole):
```
Você Sabe Quanto 
Está Perdendo?
```
4. Configurar:
   - **Font:** Inter Bold (ou Poppins Bold)
   - **Tamanho:** 52px
   - **Cor:** #FFFFFF (branco)
   - **Alinhamento:** Center
   - **Posição:** Topo-meio (acima da linha roxa)

**Passo 5: Adicione Subtexto**
1. Clique em "Texto" → "+ Adicionar texto"
2. Type:
```
Controle Total da Banca
Evite Perdas Desnecessárias
```
3. Configurar:
   - **Font:** Inter Regular
   - **Tamanho:** 24px
   - **Cor:** #06B6D4 (ciano)
   - **Alinhamento:** Center
   - **Posição:** Embaixo (abaixo da linha roxa)

**Passo 6: Revise o Layout**
- Título no topo? ✓
- Interrogações bem distribuídas? ✓
- Dinheiro caindo? ✓
- Linha roxa separando? ✓
- Subtexto legível? ✓
